# 🌍 Travelon - Travel Listing Platform

Travelon is a full-stack travel booking platform inspired by Airbnb, where users can discover destinations, create and manage listings, book stays, make secure payments, chat with an AI travel assistant, and share listings with others through a modern and responsive interface.

## ✨ Features

- 🔐 User Authentication & Authorization
- 🏡 Create, Edit & Delete Listings
- ⭐ Reviews & Ratings
- 📷 Image Upload with Cloudinary
- 💳 Secure Online Payments
- 🤖 AI Travel Chatbot
- 📅 Date Selection for Bookings
- 🔗 Share Listings
- 🗺️ Interactive Maps & Location
- ❤️ Wishlist / Favorites
- 🔍 Search & Filter Listings
- 📱 Fully Responsive Design
- ⚡ Form Validation & Flash Messages
